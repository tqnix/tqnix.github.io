HARE DOCUMENTATION:

Info ya need to know beforehand:
- Press RUN to iterate a program. Pressing RUN will not reset any values.
- Alternatively, you can use the auto-iterate function to have a program automatically iterate over and over based on a fixed amount of time.
- Press RESET to reset all values
- All sequences end with a period (.) by default.
- You can have multiple functions within a sequence. This could come in useful when using the INIT or THEN functions, or when defining a function with DEF.
- Ex. PEN DOWN MOVE 10 TURN 15 PEN UP.
- Functions with numerical parameters accept numbers, variables, or expressions enclosed in brackets.
- because the period is used for ending a sequence, you need to express non-integer numbers using fractions (i. e. 0.47 can be represented as 47/100).
- All variables are floating point numbers.
- The canvas is 256 x 256 with a gray background.
- You do not need to have quotation marks when inputting variable or function names.
- Simple (supported by JS) mathematical expressions can be entered in place of numbers by having the expression in parantheses.

VARIABLES



px: pen x position
py: pen y position
pd: pen angle (in degrees)
random: random number from 0 to 1
randominteger: random integer from 0 to 99 (inclusive)
randomangle: random integer from 0 to 359 (inclusive)

MATH SHORTHAND



These can be used in place of JS math operations, because many math functions can't be entered without a period.
You need to have parenthesis after the shorthand because the shorthand only stands in for the characters, not the parentheses.
Here is a list of the shorthand usable in Hare and their JS equivalent, or what they do:

#sin: Math.sin
#cos: Math.cos
#tan: Math.tan
#asin: Math.asin
#acos: Math.acos
#atan: Math.atan
#ceil: Math.ceil
#floor: Math.floor
#min: Math.min
#max: Math.max
#ln: Math.log
#log: Math.log10
#pi: Math.PI
#sqrt: Math.sqrt
#deg: A function that converts from radians to degrees
#abs: Math.abs

FUNCTIONS



INIT (sequence)
- Any sequence with an INIT before it will only run on the first iteration.

MOVE [number]
- Moves the pointer by that amount of pixels.

TURN [number in degrees]
- Turns the pointer by that many degrees.

PEN
- Controls various aspects about the pen
- PEN UP: Stops drawing
- PEN DOWN: Starts drawing
- PEN TOGGLE: Toggles between pen up and down
- PEN [number] adjusts width of pen

GOTO [x, y]
- Goes to a point defined by the coordinates

ANGLE [number in degrees]
- Orients the pointer to a fixed rotation in degrees.

RESET
- Restarts the program. Sets iterations to 0 and reinitializes all values.

CLEAR
- Clears the screen.

CLEAROUTPUT
- Clears the debug output at the bottom.

VAR [variable name, number]
- Defines a variable, or sets it to a certain value.
- Variables should be ideally not too long and be in letters only.

INC [variable name, number]
- Increments a variable by a certain amount. (Use negative numbers to decrement)
- see VAR for more info.

GET [variable name]
- Prints out the current value of a variable. Mainly used for debugging purposes.

ASK [variable]
- Opens a prompt for the user to enter a function.
- Alternatively, you could type !stop to stop auto iterate if it's on.

DEF [function name]
- Defines a function.
- Functions calling to other functions will crash the webpage.

CALL [function name]
- Defines a function.
- Calling to other functions will crash the webpage.

ASKFUNCTION [function name]
- Opens a prompt for the user to enter a function.
- Calling to other functions will crash the webpage.

IF [number] [comparison] [number]
- If the comparison is true, the nearest sequence preceded by a THEN is executed.

UNLESS [number] [comparison] [number]
- If the comparison is false, the nearest sequence preceded by a THEN is executed.

REPEAT [number]
- Repeats the nearest sequence preceded by a THEN is executed.

*Currently, IF, UNLESS, and REPEAT statements cannot be nested.

THEN (statement)
- Searches for the nearest IF, UNLESS, or REPEAT, and does stuff accordingly.

PRINT [text]
- Prints text.

PRINTMAFS [number]
- Prints a number.

WAIT [number]
- Waits for that amount of milliseconds. Currently very buggy.

POINTER
- Controls various aspects about the pointer
- POINTER ON: Turns pointer on
- POINTER OFF: Turns pointer off
- POINTER TOGGLE: Toggles pointer
